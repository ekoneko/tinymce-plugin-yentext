A yen text plugin for tinymce

## whats new

remove jquery depends

use es6

fix some bugs

## install

  npm i tinymce-yen

## how to use

```
    import 'tinymce-yentext'

    options = {
        toolbar1: "yen",
        plugin: "yen",
        ...
    }
    tinyMCE.init(options);
```
